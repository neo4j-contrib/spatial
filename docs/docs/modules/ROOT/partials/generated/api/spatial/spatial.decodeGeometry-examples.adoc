=== Decode a geometry from a node property

Create a WKT layer

[source,cypher]
----
CALL spatial.addWKTLayer('geom','geom')
----

.Query
[source,cypher]
----
CREATE (n:Node {geom:'POINT(4.0 5.0)'}) RETURN spatial.decodeGeometry('geom',n) AS geometry
----

.Result
[opts="header",cols="1"]
|===
|geometry
|point({x: 4.0, y: 5.0, crs: 'cartesian'})
|===
